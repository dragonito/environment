## Install

### Vim

* checkout in homedir
* ln -s environment/vimrc .vimrc
* start vim

### tmux
* in homedir
* ln -s  environment/tmux/tmux-gruvbox-dark.conf .tmux.conf

## Keybindings
leader is mapped to ","
* leader+r = NERDTreeTabToggle
* leader+t = NERDTreeTabFind
